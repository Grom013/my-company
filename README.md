# Приложение контактов

### Что я использовал:

- React
- React-Router-Dom
- Lodash

### Инструкция по запуску:

1. Скачайте приложение с GitHub:

### `https://github.com/Grom013/my-company.git`

2. Введите в терминале команду:

### `npm i`

3. Затем команду:

### `npm run dev`

### Возможности приложения:

- пагинация
- фитльтрация
- сортировка
- изменение контактов
- удаление контактов
- поиск по имени
